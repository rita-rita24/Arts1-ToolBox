# Local Notion Clone

`clone.html` はローカル完結で動作する Notion 風エディタです。

## できること
- サイドバーでページ作成・複製・削除・検索
- ページタイトル、アイコン、カバー変更
- ブロック編集（見出し、箇条書き、番号付き、To-do、引用、Callout、Code）
- `/` スラッシュコマンドでブロックタイプ切替
- Markdown ショートカット変換（`#`, `##`, `- [ ]`, `1.` など）
- ダーク/ライト切替
- `localStorage` 自動保存

## 操作メモ
- `Enter`: 次ブロック作成
- `Backspace` (空行): ブロック削除
- `Tab` / `Shift+Tab`: リスト・To-do のインデント調整
- `Ctrl+K` / `Cmd+K`: ページ検索にフォーカス
