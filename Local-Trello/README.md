# Local-Trello

ブラウザ単体で動作する、ローカル保存型の Trello 風ボードです。

## 使い方
- `Local-Trello.html` をブラウザで開く
- データはブラウザの `localStorage` に保存される
- バックアップは「ボード」ドロワー内のエクスポート/インポートを利用する

## 今回の主な修正
- インポート時の構造検証を強化し、壊れた JSON の保存を拒否
- インポート後に Undo/Redo 履歴を初期化し、旧 state 参照を防止
- 保存失敗時は直前の保存済み state へロールバックし、成功フローを中断
- `renderAll()` ごとの自動保存を廃止し、データ変更時のみ保存
- ボード名変更時のドロワー重複生成を解消
- 期限なしカードの完了操作を禁止し、完了ボタン文言を操作内容に一致
- フィルター中のカード並び替えで、非表示カード順序を維持
- 可視リスト並び替えで、アーカイブ済みリストの相対位置を維持
- トップバー操作に `aria-label` を付与し、ボード名を button 化
- カードフォーカスでもショートカット対象を取れるよう改善
- 日付ポップオーバーのローカル日付変換を修正し、UTC 正方向環境での前日化を防止

## 注意事項
- ブラウザやストレージ設定によっては保存に失敗することがあります
- 保存失敗時はトースト表示後に保存済み state を再表示します
- 重要データは定期的に JSON エクスポートしてください
