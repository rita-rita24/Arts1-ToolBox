# MustKnowCodingEnglish

Parts1〜Parts4 を 1ファイルSPAに統合した、ローカル完結の学習ツールです。

## ファイル
- `MustKnowCodingEnglish.html`: 学習・テスト・結果・設定を統合した本体

## 画面構成
- 学習: 単語カード、検索、レベル/ステータス/アルファベットフィルタ、進捗サマリー（Related Words表示なし）
- テスト: 4択問題（最大10問）、回答フィードバック、進捗バー
- 結果: 直近セッションのKPI、回答一覧、優先復習ワード
- 設定: ライト/ダーク切り替え、進捗リセット、履歴削除

## アイコン
- マテリアルアイコン（Material Symbols Rounded）をCDNから読み込み
- ヘッダー左上に `code` アイコンを表示
- `code` モチーフのSVGファビコンを設定

## 学習データ
- 収録語彙: 800語
- 前提 100語 / ベーシック 300語 / アドバンスト 300語 / 略語 70語 / 頭字語 30語

## テスト出題方針
- 4択の選択肢は「問題表示時」に1問ずつ生成（逐次生成）
- 一括で全問題分を先に作らず、現在の問題だけを評価して生成
- 同レベル・同品詞・語形近傍を優先した誤答候補を選定

## タイポグラフィ方針
- フォントは `Inter` 系を使用
- 文字間は `-0.31px` を基準に統一
- 太字強調は学習対象の英単語表示を中心に限定

## データ保存
- `localStorage` に進捗・テスト履歴・テーマを保存
- 外部通信なし（ローカル完結）

## 使い方
1. `MustKnowCodingEnglish.html` をブラウザで開く
2. 学習画面で単語ステータスを更新
3. テストを開始し、結果画面で復習対象を確認
4. 必要に応じて設定からテーマ変更やデータ管理を実施
