# MustKnowCodingEnglish

単一HTMLで動作する、基本情報技術者試験対策向けの演習特化ツールです。

## ファイル
- `MustKnowCodingEnglish.html`: 演習・成績/履歴・設定を統合した本体

## 画面構成
- 演習: 分野選択（全分野/前提/ベーシック/アドバンスト/略語/頭字語）、4択演習、即時採点、進捗表示
- 成績/履歴: 最新セッションKPI、回答一覧、直近セッション履歴（分野ラベルつき）
- 設定モーダル: ライト/ダーク切替、進捗リセット、履歴削除、全データ削除

## 演習仕様
- 検索機能と Alphabet フィルタは廃止
- 絞り込み軸は分野のみ
- 出題数は常に選択分野の全問
  - 前提: 100問
  - ベーシック: 300問
  - アドバンスト: 300問
  - 略語: 70問
  - 頭字語: 30問
  - 全分野: 800問
- 4択の選択肢は問題表示時に逐次生成
- 正解位置はセッション単位シードでシャッフル
- 回答後は即時採点され、選択肢はロックされる
- フィードバックは「正解」と「意味（要点）」を表示

## データ保存
- `localStorage` の保存キーは `must-know-coding-english-state-v2`
- 初回起動時に旧 `v1` データがあれば「引き継ぐ/初期化」を選択
- 保存不可環境では一時利用モードで動作し、フッターに保存状態を表示
- 外部通信なし（ローカル完結）

## 使い方
1. `MustKnowCodingEnglish.html` をブラウザで開く
2. 演習画面で分野を選択して開始
3. 1問ずつ解答し、即時フィードバックを確認
4. 成績/履歴画面で最新結果と直近履歴を確認
5. 必要に応じて設定からテーマ変更やデータ管理を実行
