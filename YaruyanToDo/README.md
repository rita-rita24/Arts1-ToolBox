# Yaruyan! Todo

ローカル保存だけで動く、1ファイル完結の Todo ツールです。  
「追加する → 手を付ける → 完了する」を短い導線で回せることを重視しています。

## このツールで得られる価値
- 親タスクと子タスクを同時に扱えて、今日やることの全体像が崩れにくい
- 集中モードで着手と完了を明確にし、実績時間を残せる
- 分析ページで進捗（完了率・残件）がすぐ分かる
- 設定でテーマや演出を変えられ、続けるモチベーションを保ちやすい

## 1分クイックスタート
1. `yaruyanTodo.html` をブラウザで開く
2. Todo リストの空状態で `お試しタスクを追加` を押す
3. 追加されたタスクの開始ボタンを押して、完了まで一度体験する
4. 自分のタスクを入力して運用を開始する

## 主な機能
- 親タスク / 子タスク管理（ドラッグで並び替え対応）
- 集中モード（開始 / 一時停止 / 完了 / 中断）
- 完了演出（効果音・お祝い表示）
- 整理メモ（ローカル保存）
- 分析ダッシュボード（件数・完了率）

## 保存仕様
- データはブラウザの `localStorage` に保存されます
- 通信先サーバーは使わず、端末内で完結します
- ブラウザデータを削除すると保存内容も消えるため、共有端末ではご注意ください
