# やるやん！Todo

ローカル保存だけで動く、1ファイル完結の タスク ツールです。
「追加する → 手を付ける → 完了する」を短い導線で回せることを重視しています。

## このツールで得られる価値
- 親タスクと子タスクを同時に扱えて、今日やることの全体像が崩れにくい
- 集中モードで着手と完了を明確にし、実績時間を残せる（ポモドーロ休憩時間は含まない）
- タスク画面ヘッダー右上から、完了済み削除・全削除をすぐ実行できる
- 設定でテーマや演出を変えられ、続けるモチベーションを保ちやすい

## 1分クイックスタート
1. `YaruyanTodo.html` をブラウザで開く
2. タスク リストの `タスクを追加` 入力欄に、例として `見積もりを作る` と入力する
3. 入力欄右側の `追加` ボタンを押す
4. 追加されたタスクの `開始` ボタンを押して、完了まで一度体験する
5. 自分のタスクを入力して運用を開始する

## アプリ本体の体験手順
1. `YaruyanTodo.html` をブラウザで開く
2. タスク リストの `タスクを追加` 入力欄に、例として `見積もりを作る` と入力する
3. 入力欄右側の `追加` ボタンを押す
4. 追加されたタスクの `開始` ボタンを押して、完了まで一度体験する
5. 自分のタスクを入力して運用を開始する

## 主な機能
- 親タスク / 子タスク管理（ドラッグと `Alt+ArrowUp/Down` で並び替え対応）
- タスク一覧の表示フィルタ（すべて / 未完了 / 完了、選択状態は再読込後も保持）
- 集中モード（開始 / 一時停止 / 完了 / 中断）
- 集中モードは再読込時に一時停止として復元（閉じていた時間は実績に加算しない）
- ポモドーロ（設定で集中/休憩の分数を指定、休憩時間は実績時間に含まない）
- 完了演出（効果音・お祝い表示）
- テーマカラー選択（左端はエメラルド）
- 設定画面のテーマカラーは見出しラベルなしでスウォッチ一覧から選択
- テーマカラー選択は `radiogroup` として矢印キー / Home / End / Enter / Space に対応
- タスク追加ボタンと各種トグルの ON 色は主アクション色（テーマ主色）を使用
- ライト/ダークモード切り替えボタンは、ライト時のみ青空イメージの固定ブルーを使用
- 整理メモ（ローカル保存）
- 整理メモの `textarea` は初期表示時に縦方向を大きく確保（画面高に応じて自動調整）
- タスク画面ヘッダー右上の一括削除ボタン（完了済み / すべて）
- 削除確認ダイアログで対象名・件数を表示
- 削除 / 一括削除は実行後5秒以内なら「元に戻す」で復元可能（通常トースト表示の影響を受けない独立表示）
- 削除Undoは削除差分のみ復元し、削除後に行った別編集は巻き戻さない
- 個別削除（親/子）は待機なしで即時反映
- 完了取り消しは実績時間（経過時間）を保持したまま `done` のみ戻す
- `undo` アイコンは全表示箇所で縦位置を微調整し、他アイコンとの視覚的なセンターを揃えています
- 集中モード中断時の確認ダイアログ（実績時間の破棄確認）
- アイコンのみのボタンに即時表示される独自ツールチップ（ホバー / キーボードフォーカス時、上優先で収まらない場合は下へ切替）
- 使用アイコンはHTML内SVGスプライトで内製化（外部フォント不要）
- タスク一覧は見出し行なしのシンプル表示
- アクセシビリティ補強（入力ヒントの読み上げ、設定トグルのラベル関連付け、モーダルのフォーカストラップと復帰、確認ダイアログのボタン別 Enter 操作）
- モーダル（編集/確認/集中）の表示・非表示は即時切り替え（アニメーションなし）

## 保存仕様
- データはブラウザの `indexedDB` を主保存先として保存されます
- 既存の `localStorage` データがある場合は、起動時に `indexedDB` へ自動移行されます
- 表示フィルタ状態（`すべて / 未完了 / 完了`）も保存されます
- 通信先サーバーは使わず、端末内で完結します
- `localStorage` が失敗しても `indexedDB` 保存が成功している場合は警告を固定表示しません
- 集中モード実行中に再読込した場合は、直前までの経過時間のみ保存して一時停止状態で復元されます
- ブラウザデータを削除すると保存内容も消えるため、共有端末ではご注意ください
