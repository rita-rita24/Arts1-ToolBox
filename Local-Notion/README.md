# Local Notion Clone

`clone.html` はローカル完結で動作する Notion 風エディタです。

## できること
- サイドバーでページ作成・複製・削除・検索
- ページタイトル、アイコン、カバー変更
- ブロック編集（見出し、箇条書き、番号付き、To-do、引用、Callout、Code）
- 入力欄は1つだけで、Notion風に先頭記法でブロックへ自動変換
- `/` スラッシュコマンドでブロックタイプ切替
- Markdown ショートカット変換（`#`, `##`, `- [ ]`, `1.` など）
- ダーク/ライト切替
- `localStorage` 自動保存

## 操作メモ
- `Enter`: 次ブロック作成
- `Backspace` (空行): ブロック削除
- `Ctrl+K` / `Cmd+K`: ページ検索にフォーカス
- `#` + `Space`: 見出し1へ変換（`##`, `###` も同様）
- `-` + `Space` / `1.` + `Space`: リストへ変換
- `"` + `Space` / `>` + `Space`: 引用ブロックへ変換
