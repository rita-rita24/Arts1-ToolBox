# Local Notion

`LocalNotion.html` は、単一 HTML ファイルで動作するローカル向け Notion 風エディタです。

## 起動方法
1. `LocalNotion.html` をブラウザで開きます。
2. そのまま編集を開始できます（インストール不要）。

## 主な機能
- サイドバーでページの作成・複製・階層管理
- ページのゴミ箱移動・復元・完全削除
- ブロック編集（見出し、箇条書き、番号付き、ToDo、引用、コールアウト、コード、区切り線、テーブル、画像）
- ブロック左の `+` ボタンから、追加するブロックタイプを選択して挿入
- ブロック選択メニュー（`+` / `/` / タイプ変換）は共通順序のコンパクト表示
- `/` スラッシュコマンドと Markdown ショートカット
- 検索モーダル（`Cmd/Ctrl + K`）
- ライト/ダーク切替とテーマカスタマイズ（アクセント色、4種類のフォント: Noto Sans JP / Meiryo / Yu Mincho / Code Mono、ページ幅）
- Markdown インポート / エクスポート
- 保存（アプリ全体を 1 ファイルの HTML として書き出し）

## 保存仕様
- メイン保存先は `IndexedDB`（自動保存）です。
- `IndexedDB` 保存に失敗した場合は `localStorage`（キー: `notion-clone-v4`）に退避します。
- サイドバー幅は `localStorage`（キー: `localnotion_sidebar_width`）に保存し、読込不可な環境では既定幅にフォールバックします。
- HTML 書き出し時は、現在状態を `embedded-state` に埋め込みます。

## ゴミ箱と自動削除
- ページ削除はまずゴミ箱への移動（ソフト削除）です。
- 保持日数（0 / 7 / 30 / 90日）を設定すると、期限切れページは自動で完全削除されます。
- 自動削除が実行された場合は件数通知を表示し、ゴミ箱モーダル内にも直近実行履歴を表示します。
- 完全削除は元に戻せません。

## テーマ
- テーマ切替時に `data-theme` と `document.documentElement.style.colorScheme` の両方を更新します。
- ネイティブの `input` / `select` / `color` などもテーマと同期します。

## 主要ショートカット
- `Enter`: 次ブロック作成
- `Shift + Enter`: ブロック内改行
- `Backspace`（空行）: ブロック削除
- `Cmd/Ctrl + K`: 検索モーダル
- `Cmd/Ctrl + Z`: 元に戻す
- `Cmd/Ctrl + Shift + Z` または `Cmd/Ctrl + Y`: やり直し

## 既知の注意点
- アイコンフォントとコードハイライトは外部 CDN（Google Fonts / Prism）を利用しています。
- ネットワーク制限環境では、見た目やコードハイライトが一部低下する場合があります。
