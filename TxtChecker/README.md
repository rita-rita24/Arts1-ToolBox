# 文字品質チェッカー Lite

単一HTMLで動く、テキスト品質チェック用ツールです。  
全角数字・全角英字・全角スペース・タブ・半角カナ・カスタムNG文字を検出し、対応項目は自動修正できます。

## 2026-04-01 保守改善

### 変更内容
- クリップボードAPIが使えない環境でも、フォールバックコピーで結果/テキストをコピー可能に改善
- `matchMedia().addEventListener` 非対応ブラウザで起動エラーにならないよう互換処理を追加
- `localStorage` 保存失敗時の例外を握りつぶさず、UI通知して処理継続するよう改善

### 影響
- 既存UI/既存機能の公開仕様は維持
- 一部ブラウザや制限環境で、コピー・テーマ反映・設定保存の失敗耐性が向上
