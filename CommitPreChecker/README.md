# CommitPreChecker

## Files
- `commitPreChecer-Admin.html`: 管理者向け。設定編集、JSON入出力、配布用HTML生成を行う単一HTML。

## Current Scope
- 右カラムなしの単一カラム管理画面
- Admin画面はライト基調で統一（黒背景セクションなし）
- タイポグラフィは基本14px基準（強調箇所のみ段階的に拡大）
- Claude（Anthropic）デザイントークン準拠のUI（暖色ニュートラル、セリフ見出し、リングシャドウ）
- 基本情報（ルールセット名・説明・ステータス）
- 言語非依存の共通ルール設定
- チェック観点ごとの重大度設定（ID表示なし）
- 配布オプション（ルール表示可否 / 自動修正可否 / テーマ / 有効期限）
- 配布用単一HTML生成（`embedded-config` JSON埋め込み）
- 設定JSONの保存 / 読込

## Generated HTML Behavior
- ツール名: `CommitPreChecker`
- 外部通信なし（ローカル実行前提）
- Claude（Anthropic）デザイントークン準拠の配色・タイポ・操作系UI
- 同一テーマ内で明暗を混在させない配色（ライトはライト基調、ダークはダーク基調）
- 生成日時は画面表示しない
- タイポグラフィは基本14px基準（強調箇所のみ段階的に拡大）
- 入力コードチェック
- 問題一覧表示（チェック観点名 / 行番号 / 重大度 / 修正案）
- 安全な自動修正
- 修正後コードコピー
- ルールモーダル、ヘルプモーダル
- 背景透かしテキストなし

## Implemented Checks (Language-Agnostic)
- インデント方式の混在
- インデント幅の不一致
- タブは表示桁（タブ幅）換算で判定
- 継続行らしい行（演算子連結・括弧継続など）は幅不一致の誤検知を抑制
- 行末空白
- 改行コードの不一致
- ファイル末尾改行
- 全角スペース
- 不可視文字
- 連続空行の超過
- BOM混入
- 行長超過

## Auto Fix Policy
- 対応: 行末空白除去、末尾改行調整、改行コード統一、連続空行削減、タブ/スペース単純置換
- 非対応: 意味変更リスクが高い補正（文脈依存の構文補正、文字列内部補正）

## Usage
1. `commitPreChecer-Admin.html` をブラウザで開く
2. ルールと配布設定を入力
3. `配布用HTMLを生成 (Generate HTML)` を実行
4. ダウンロードされたHTMLを利用者へ配布
