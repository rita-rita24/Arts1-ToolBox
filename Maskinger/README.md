# Maskinger

AIに入力する前のテキストを、ブラウザ内で自動マスキングする単体HTMLツールです。

## できること

- 入力欄に打ち込んだ瞬間に、右側へリアルタイムでマスキング結果を表示
- 左右2ペインの最小レイアウト（左:入力 / 右:マスク結果）
- 通常テキストの機密情報をマスク
  - メールアドレス
  - URL
  - IPアドレス
  - 電話番号
  - SSN 形式 (`123-45-6789`)
  - 郵便番号 (`〒123-4567` / `123-4567`)
  - IBAN 形式
  - UUID
  - クレジットカード番号（Luhnチェックで妥当性確認）
  - 12桁の個人番号形式
  - ローカルパス
  - 秘密情報っぽい値 (`password`, `token`, `api_key` など)
  - PII系キーへの代入値 (`email`, `phone`, `address`, `account_number`, `user_id` など)
  - 秘密鍵ブロック (`-----BEGIN ... PRIVATE KEY-----`)
  - Bearer トークン
  - 認証情報入り接続文字列 (`postgres://user:pass@...` など)
  - 主要API/アクセストークン形式（OpenAI, GitHub, Anthropic, Google, Stripe, Slack, AWSなど）
  - 敬称付き氏名、会社名らしき文字列
- コードらしい入力の場合は追加で自動処理
  - コメントを削除 (`//`, `/* */`, `#`, `--`)
  - SQL文に見える文字列リテラルを `[SQL_n]` に置換
  - 定数名（`UPPER_SNAKE_CASE` など）を `[CONST_n]` に置換
- 出力欄の初期表示は空（入力後にのみ結果を表示）

## 使い方

1. `Maskinger.html` をブラウザで開きます。
2. 入力欄にテキストまたはコードを貼り付けます。
3. 右側の出力欄にリアルタイム反映された結果を確認します。
4. 「コピー」ボタンで結果をAIに貼り付けます。

## 補足

- 変換はブラウザ内で完結し、外部送信は行いません。
- マスキングはルールベースです。文脈依存の固有名詞は取りこぼす場合があるため、送信前の最終確認を推奨します。
