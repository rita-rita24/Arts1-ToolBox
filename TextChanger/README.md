# TextChanger

1行ごとの入力値を、5種類の形式へ同時変換する単体HTMLツールです。

## できること

- 画面を左右1:2で表示（両端に少し余白あり）
  - 左: 入力欄のみ
  - 右: 変換結果カード5種を縦に表示
- 入力に応じて、5モードを同時リアルタイム変換
  - A5M2用 Caption 変換
  - タブ区切り変換（Excel向け）
  - SQL の IN 句変換
  - PostgreSQL `SELECT + INSERT + DELETE` 変換
  - 連番調整（テキスト中の数値を一括で増減）
- 各カード右上の「コピー」ボタンで、そのカード結果だけをコピー
- ライト/ダークモードをヘッダー右上のスイッチで切り替え
- 入力内容とテーマ設定を `localStorage` に保存

## 使い方

1. `TextChanger.html` をブラウザで開きます。
2. 左の入力欄に、1行に1つずつ値を入力します。
3. 右側4カードに同時表示された結果を確認します。
4. 必要な形式のカード右上「コピー」を押して利用します。

PostgreSQL `SELECT + INSERT + DELETE` カードの使い方:
- 1行目をカラム名（ヘッダー）、2行目以降をデータ行にしたタブ区切り表を貼り付けます（Excelのコピー貼り付けを想定）。
- コピーボタン左のテーブル名入力で、生成先テーブルを変更できます（初期値は空欄）。
- テーブル名入力を変更すると、生成結果に即時反映されます。
- テーブル名入力が空欄の場合は、`???` で生成します。
- 空欄セルと `NULL` 文字列（大文字小文字無視）は `NULL` として扱います。
- 事前チェック用の `SELECT` 文を冒頭に出力します（各行の `DELETE` と同じ条件で `SELECT * FROM ...`）。
- `INSERT` と `DELETE` はそれぞれ `BEGIN;` / `COMMIT;` で囲んで生成します。

連番調整カードの使い方:
- コピーボタン左の増減値入力に整数を入れると、入力テキスト中のすべての数字に対して足し算（負数なら引き算）した結果が出力されます。
- 入力の改行や空行はそのまま保持されます。例えば `No.1` → `No.2` → `No.3` のような連番テキストに対して `5` を入れると `No.6` → `No.7` → `No.8` になります。
- 増減値が空欄または `0` の場合は、入力テキストをそのまま出力します。

## 補足

- 外部送信は行わず、変換処理はブラウザ内で完結します。
