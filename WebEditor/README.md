# WEW (Web Editor Web)

`wew.html` は、ローカルで動作する単一HTMLのWebエディタです。

## このフォルダの使い方

1. `wew.html` をブラウザで開く
2. 新規作成 / 開く / 保存 / 実行を上部メニューで操作する

## 日本向けリファクタリング内容

- UI文言を日本語化（新規/保存/リネーム/ヘルプ/設定/About）
- 新規HTMLテンプレートに `lang="ja"` を追加
- 設定ダイアログ・ショートカット説明を日本語化
- `document.documentElement.style.colorScheme` をテーマ連動で反映
- 外部CDNのLint読込を削除し、Lintはライブラリ存在時のみ有効化
- メニュー/ファイル種別アイコンを Material Icons（CDN）へ置換
- Googleドキュメント風のUIへリデザイン（ライトグレーキャンバス + 白いツールバー/編集面 + 青アクセント）
- Roboto + Noto Sans JP ベースのタイポグラフィへ変更
- ツールバー/ダイアログの Material Icons をボタン中央に揃えるようスタイル修正
- エディタの初期フォント倍率を `0.95rem` へ調整し、設定から `0.75` 以上で細かく変更可能
- `setTheme()` のUI色強制上書きを整理し、エディタテーマ切替とデザイン一貫性を両立
- メニュー表示/非表示ボタンとF10トグルを削除し、上部メニューを常時表示に変更

## 補足

- このツールはローカルファーストで動作します。
- 外部ネットワーク通信は行いません（Material Icons CDN 読込と、ユーザーが明示的に外部リンクを開く場合を除く）。
