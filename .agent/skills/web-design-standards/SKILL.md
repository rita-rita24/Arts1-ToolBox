# Web Design Standards

You MUST follow the guidelines and standards defined in the resource files below when designing, refactoring, or reviewing web applications.

## Key Resources

- **UI Baseline**: Read [baseline-ui.md](resources/baseline-ui.md) for visual design standards.
- **Accessibility**: Follow [fixing-accessibility.md](resources/fixing-accessibility.md) for a11y compliance.
- **Motion & Performance**: Refer to [fixing-motion-performance.md](resources/fixing-motion-performance.md).
- **Metadata**: Use [fixing-metadata.md](resources/fixing-metadata.md) for SEO and meta tags.

## Instructions

1. **Before writing code**: Check `baseline-ui.md` to ensure design consistency.
2. **Reviewing code**: Use the checklists in these documents to validate implementation.
